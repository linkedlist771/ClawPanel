# ClawPanel v5.0.6 更新日志

发布日期：2026-02-27

## 🔧 重大修复

### 全新服务器一键部署全流程修复
- **Node.js 安装增强**：NodeSource 失败时自动下载 Node.js v22 官方二进制 tarball，支持 OpenCloudOS/TencentOS 等 RHEL 兼容系统
- **NapCat 安装自动装 Docker**：安装 NapCat 时如未检测到 Docker，自动通过阿里云镜像安装 Docker CE
- **Docker/Node.js 发行版检测改进**：使用 `/etc/os-release` 替代 `command -v` 检测，支持更多 Linux 发行版
- **NapCat QR码 Unauthorized 修复**：默认 WebUI Token 与安装脚本一致（`clawpanel-qq`），扫码登录不再报 Unauthorized

### OpenClaw 进程管理修复
- **自动启动 OpenClaw**：ClawPanel 启动时自动运行 `openclaw gateway`，无需手动启动
- **重启网关自动启动**：点击"重启网关"时，如 OpenClaw 未运行会自动启动，不再仅写信号文件
- **进程管理器命令修复**：`openclaw start` → `openclaw gateway`（适配 OpenClaw 2026.2.26+）

### OpenClaw 安装后自动配置
- **自动安装 QQ 插件**：一键安装 OpenClaw 后自动下载并安装 QQ (OneBot11) 通道插件
- **自动初始化配置**：自动设置 `gateway.mode=local`、`channels.qq.wsUrl`、`plugins.entries.qq` 等关键配置

### 面板自更新修复
- **面板更新功能重写**：使用外部 Shell 脚本执行更新（停止服务→替换二进制→启动服务），解决运行中二进制无法替换和自重启竞态问题
- **Windows 更新兼容**：Windows 使用 rename 方式替换，避免文件锁定

## 📋 技术细节
- 支持的 Linux 发行版：Debian/Ubuntu、CentOS/RHEL/Fedora/Rocky/Alma、OpenCloudOS、TencentOS
- Node.js 安装优先级：NodeSource → 系统包管理器 → 官方二进制 tarball (v22 LTS)
- Docker 安装源：阿里云镜像 (mirrors.aliyun.com/docker-ce)
